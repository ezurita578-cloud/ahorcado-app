# Ahorcado en Go

Este es mi juego del ahorcado hecho en Go para la materia de Estructura de Datos.

## Qué necesitas

- Tener Go instalado (yo use la version 1.25.7)
- Windows

## Como correrlo

Primero clona el repo:
git clone https://github.com/ezurita578-cloud/ahorcado-app

Entra a la carpeta:
cd ahorcado-app


Y correlo con:
go run hangman.go


## Como se juega

- El juego elige una palabra al azar
- Cada palabra tiene una pista para ayudarte
- Tu vas diciendo letras una por una
- Si la letra esta en la palabra aparece, si no, se dibuja una parte del muñeco
- Tienes 6 intentos en total
- O adivinas la palabra o te ahorcan jaja
